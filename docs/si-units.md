# 国际单位制 SI：基本单位

国际单位制（SI）有七个基本单位。它们不是彼此换算出来的普通单位，而是现代测量体系的基础：长度、质量、时间、电流、热力学温度、物质的量和发光强度都从这里出发。

| 物理量 | SI 基本单位 | 符号 |
| --- | --- | --- |
| 长度 | 米 | m |
| 质量 | 千克 | kg |
| 时间 | 秒 | s |
| 电流 | 安培 | A |
| 热力学温度 | 开尔文 | K |
| 物质的量 | 摩尔 | mol |
| 发光强度 | 坎德拉 | cd |

本文先整理长度基本单位 **米（meter / metre）**，作为 MeasureQuest 后续制作 SI 单位卡片、章节和网站页面模块的基础。

## 米（meter / metre）

### 单位卡片草稿

### meter 与 metre

**meter** 和 **metre** 都表示“米”这个长度单位。

- **meter** 多见于美式英语；
- **metre** 多见于英式英语，也常出现在国际计量组织和历史文献中；
- 两种拼写指向同一个 SI 长度基本单位，符号都是 **m**。

MeasureQuest 可以在面向读者的标题中同时写作“米 / meter / metre”，帮助不同英语背景的读者识别同一概念。

### 历史来源：法国大革命、公制改革与地球子午线

米的诞生与 18 世纪末的法国大革命和公制改革紧密相关。

在旧制度下，欧洲各地存在大量地方单位。相同名称在不同城市可能代表不同长度，贸易、税收、工程和科学交流都容易出现混乱。法国革命者希望建立一套更理性、更统一、更公平的度量衡制度，于是提出用自然尺度来定义新的长度单位。

最初的米被设想为：

> 从北极经过巴黎子午线到赤道距离的一千万分之一。

这个定义让米不再依附于某个国王的身体、某个城市的习惯或某个行业的传统，而是试图把长度标准建立在地球本身之上。

更完整的叙事可见 MeasureQuest 核心文章：[《一米的故事：从地球子午线到光速》](meter-history.md)。

### 从米原器到国际原型米尺

地球子午线定义具有强烈的理想色彩，但日常校准仍然需要可保存、可复制的实物标准。

- **1799 年**，法国制作了代表一米长度的铂金标准尺，也就是早期的米原器。
- **1889 年**，国际原型米尺被正式采用。它由铂铱合金制成，稳定性优于早期铂金标准尺，成为国际长度标准的重要实物基础。

这些金属标准的优点是直观：人们可以保存它、复制它，并用它校准其他尺子。缺点也很明显：任何实物都可能受到温度、磨损、污染、材料变化和保存条件影响。对于越来越精密的科学测量来说，“一根尺子”本身不够稳定。

### 1960 年：用氪原子光波长定义米

20 世纪，光谱学的发展让科学家看到新的长度标准。

1960 年，米被重新定义为氪-86 原子某一特定光谱线在真空中波长的 **1,650,763.73 倍**。

这意味着米不再主要依赖保存在某个地点的一根金属尺，而是依赖可在实验室复现的原子光谱现象。只要实验条件足够严格，不同地方的科学家都可以用同一种自然现象复现长度标准。

### 1983 年：用光速定义现代的米

1983 年，米被重新定义为：

> 1 米是光在真空中于 1/299,792,458 秒内传播的距离。

这个定义沿用至今。它的关键思想是：把真空中的光速固定为精确值，再结合已经能够极高精度定义的“秒”，由时间和光速共同确定长度。

这也回答了 MeasureQuest 的核心问题：

> 一米为什么不再是一根尺子，而是由光速定义？

因为实物标准会老化、受环境影响，也难以在全球范围内以完全相同的方式复制；而光速这样的基本物理常数更稳定、更普遍，更适合作为现代科学和工程共享的标准。今天的米不是某根金属条的长度，而是自然规律中可以被精确复现的距离。

### 米的现代应用

米是日常生活、工程、科学和导航中的基础语言。

- **日常生活**：房间尺寸、身高、家具、道路距离和运动成绩都常用米或由米派生出的厘米、千米表示。
- **工程建设**：建筑、桥梁、铁路、机械制造和城市规划都依赖统一、精确的长度标准。
- **科学研究**：物理实验、天文观测、材料科学和生命科学需要可复现的长度测量。
- **导航与定位**：地图、测绘、卫星导航和地理信息系统都离不开以米为基础的距离表达。

从法国大革命时期的公制理想到现代光速定义，米的历史展示了 MeasureQuest 的主线：人类把测量标准从身体经验、地方习惯和实物器具，逐步推进到可共享、可复现的自然常数。
