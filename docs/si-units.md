# 国际单位制 SI：开尔文

## 开尔文 / kelvin

开尔文（kelvin，符号 **K**）是国际单位制 SI 中的**热力学温度基本单位**。它用于描述温度本身，而不只是日常冷热感受，因此在科学和工程中比摄氏度更适合作为统一的温度尺度。

摄氏度（°C）以日常经验中的水的冰点和沸点为重要参考；开尔文则从**绝对零度**开始计量。两者的刻度间隔大小相同：温度升高 1 K，与升高 1 °C 表示相同的温度差。它们的换算关系是：

```text
T(K) = t(°C) + 273.15
t(°C) = T(K) - 273.15
```

因此，0 °C 等于 273.15 K，100 °C 等于 373.15 K。书写开尔文时通常只写 `K`，不写度数符号。

## 为什么从绝对零度开始？

绝对零度是热力学温度的最低点，定义为 **0 K**，也就是 **-273.15 °C**。在这个尺度上，温度不再依赖某种日常参照物，而是与物质内部粒子的热运动和能量有关。

科学中的温度要从绝对零度开始，是因为热力学温度描述的是一个系统拥有多少热能、粒子热运动有多强。绝对零度提供了一个自然的零点：当温度接近 0 K 时，系统的热运动趋向最低可能状态。这样一来，10 K、100 K、1000 K 都能被理解为相对于同一个自然零点的温度，而不是相对于水的冰点这样的约定参考。

这种从绝对零度开始的尺度，让许多物理公式更简洁。例如温度可以直接作为能量尺度使用，表示热运动、热辐射、气体状态和相变行为。

## 名称来源

“kelvin”这个名称来自英国物理学家 **Lord Kelvin（开尔文勋爵，William Thomson）**。他在 19 世纪研究热力学和绝对温标，推动了以绝对零度为起点的温度尺度。

## 2019 年后的现代定义

2019 年国际单位制重新定义后，开尔文不再依赖某种特定物质或实验条件，而是通过固定**玻尔兹曼常数**（Boltzmann constant，符号 **k**）的数值来定义。

玻尔兹曼常数把温度和微观粒子的能量联系起来。它说明：温度不是孤立的数字，而是可以与物质内部粒子的平均热运动能量对应起来的物理量。2019 年后，玻尔兹曼常数的精确数值被固定为：

```text
k = 1.380649 × 10^-23 J/K
```

这体现了现代 SI 的核心思想：重要单位尽量基于自然常数，而不是依赖某个实物器具或某种容易受环境影响的物质状态。

## 科学与技术中的应用

开尔文广泛用于需要精确、统一温度尺度的领域：

- **物理学**：描述热力学、气体状态、黑体辐射、低温物理和相变。
- **化学**：计算反应速率、平衡常数、气体体积和热力学性质。
- **气候科学**：分析地球能量收支、辐射温度和气候模型中的温度变化。
- **低温技术**：研究液氮、液氦、超导、量子设备和极低温实验。
- **天文学**：描述恒星表面温度、宇宙微波背景辐射和星际气体温度。

## 日常温度换算示例

| 场景 | 摄氏温度 | 开尔文温度 |
| --- | ---: | ---: |
| 水的冰点 | 0 °C | 273.15 K |
| 常见室温 | 20 °C | 293.15 K |
| 人体正常体温附近 | 37 °C | 310.15 K |
| 水的沸点（标准大气压） | 100 °C | 373.15 K |
