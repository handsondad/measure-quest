# MeasureQuest 参考资料库

本文档用于集中管理 MeasureQuest 后续内容创作可参考的文章、机构页面、书籍、访谈和视觉素材来源。新增资料时，请优先选择权威、稳定、可长期引用的来源，并按照下方固定格式记录。

## 使用格式

```markdown
- [资料标题](链接)
  - 简介：用 1-2 句话说明资料内容和可用价值。
  - 适用章节：一米的故事 / SI 单位 / 古代文明 / 其他章节。
  - 素材类型：文章 / 官方资料 / 时间线 / 百科 / 书籍 / 视频访谈 / 图片素材。
```

## 章节标签说明

- **一米的故事**：适合讲述米的起源、法国大革命、公制改革、米原器、《米制公约》和光速定义。
- **SI 单位**：适合讲述公制系统、国际单位制、BIPM、现代单位定义和自然常数。
- **古代文明**：适合讲述古埃及、古巴比伦、古中国、古罗马等文明中的早期度量衡。
- **其他章节**：适合标准化、国家治理、贸易、现代测量、未来测量或视觉设计等内容。
